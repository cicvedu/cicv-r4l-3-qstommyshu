Q：作业5中的字符设备/dev/cicv是怎么创建的？
答: R4L库里的代码创建的/dev/cicv
`struct RustChrdev {
    _dev: Pin<Box<chrdev::Registration<2>>>,
}
`

它的设备号是多少？

答: cicv 设备号:  248, 0
![alt text](image-14.png)

它是如何与我们写的字符设备驱动关联上的？
答: 创建字符设备的时候，/dev 下会生成一个设备文件，就是cicv, 这样就跟设备驱动关联上了
